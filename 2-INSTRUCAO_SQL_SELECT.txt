A instrução SQL SELECT:

    Essa instrução é usada para pegar informações do Banco de Dados.

    Sintaxe:
    
        SELECT coluna1, coluna2, ... FROM nomeDaTabela;
    :

    Exemplo:
    
        SELECT * FROM Clientes;

            SELECT   = Pegue

            *        = Todas as colunas

            FROM     = Da tabela

            Clientes = Nome da tabela

        ===================================

        SELECT Nome, Idade FROM Clientes;

            SELECT      = Pegue

            Nome, Idade = As colunas Nome e Idade

            FROM        = Da tabela

            Clientes    = Nome da tabela
    :
